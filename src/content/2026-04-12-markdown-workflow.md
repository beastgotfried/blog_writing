Markdown posts let me write quickly without touching layout code.

My lightweight workflow:

1. Create a new `.md` file with frontmatter.
2. Add title, date, tags, and a short excerpt.
3. Keep each note focused on one learning.

A simple content pipeline is enough while the project is young.
